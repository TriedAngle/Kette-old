# Frozen Forth
forth implementation in x64 assembly

## 
assembly =(compiler)=> forth =(interpreter) => common lisp =(compiler)=> ice
## Notes
### Token Format
<bytes>
1 = opcode (max: 256) 
4 = line
4 = char in line
<...> data
[add, sub, mul, div] = none
[dump] = none
[push] = 8 # number | address (TODO later)

2. rsi
3. rdx
4. rcx
5. r8
6. r9
7. stack

Return:

2. rsp

6. r14
